!THIS IS THE DEVELOPMENTAL BUILD! 

To run the application:
1. Double-click the application file "ToDo_App"
2. If the app fails to run, check if the specified port is available (e.g. 7121)

NOTE: 
1. Make sure you do NOT close the command console to keep the server running
2. Setting the deadline is optional, but if you decide to set it, setting the time will also then be required

To close the application:
Simply close the browser AND the command console. Or simply close the command console to reset the server.

Error logs are saved under the folder ToDoApp_Logs
